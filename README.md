# EPOCH4.0-SQooler

Welcome to the SQL Repository of EPOCH 4.0! 

## How to contribute?

1. Go through the image attached for each schema.
2. Create a folder under the same problem name that must include the following:
     - problem statement
     - output-image file
     - solution-sql file.
4. For inspo, you can use [reference](https://github.com/github-community-gitam/EPOCH4.0-SQooler/tree/main/Student/problem_03).
  
Make sure to go through the [guidelines](https://github.com/github-community-gitam/EPOCH4.0-SQooler/blob/main/CONTRIBUTING.md).
